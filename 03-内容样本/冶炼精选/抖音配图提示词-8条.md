# 8条抖音脚本配图提示词

---

## 1. NPC识别
```
抖音口播封面：一个面无表情的人脸正面特写，脸部有微弱的像素化/马赛克效果，暗示"被编程的"。深色背景，左上方有一行发光代码文字"N-P-C"。冷色调+金色瞳孔光。科幻感但不恐怖，高认知知识博主风格。16:9。
```

## 2. 人情即账本
```
视觉概念图：一本半开的古老皮质账本漂浮在深色空间中，账本上不是数字而是人脸侧影。金色光芒从账本缝隙中透出。背景隐约可见天平。学术暗金风格，意境深邃。16:9。
```

## 3. 信息封锁
```
概念图：一个人物侧影站在一面巨大的透明幕墙前，幕墙外是模糊的城市轮廓。幕墙上有些文字被刻意涂黑遮蔽。光线从幕墙缝隙照入形成光束。强烈的明暗对比，冷蓝+金色。隐喻"信息封锁感"。16:9。
```

## 4. 多巴胺劫持
```
科学风格概念图：人类大脑侧视图，多巴胺分子以发光金色粒子形式在大脑奖赏回路中流动——形成一条过山车轨道般的曲线。轨道下方是暗色的"基线"。画面传递"被化学物质驱动"的感觉。深色背景，科技感但不过于医学化。16:9。
```

## 5. 控制二分法
```
极简哲学风：深色背景中央，一道垂直金色光线将画面分为左右两半。左边有一个张开的手掌（象征掌控），右边有一个松开的手掌（象征放下）。两个手势是同一只手的不同状态。沉静、智性、不鸡汤。16:9。
```

## 6. 四大永恒人性
```
概念图：深色背景上四个并排的古代面具——贪婪的黄金面具、魅惑的红色面具、恐惧的骷髅面具、饥渴的青铜面具。每个面具下面对应一行发光的现代符号($、❤️、💊、💳)。暗金学术风格，博物馆展陈感。16:9。
```

## 7. 强势文化
```
极简肖像图：一个人物正面半身像，站立姿势，身体周围有一圈薄薄的金色光晕边界。他/她身后是模糊的围观人群，但人物本人视线上扬，仿佛在看更远的地方。传递"不被外界定义"的感觉。深灰+金色点缀。16:9。
```

## 8. 概率思维
```
数学美学概念图：一个巨大的骰子悬浮在深色空间中，骰子的面上不是数字——是分支路径（决策树）。周围漂浮着贝叶斯公式的发光数学符号。有一种"宇宙是概率的"震撼感。暗蓝+金色公式文字。16:9。
```
