# GEO服务执行手册：从接单到交付

> 不是理论——是你可以直接照着做的流程。

---

## 第一步：GEO诊断（2-3小时）

### 1.1 搜索基线（30分钟）

打开以下平台，搜客户品牌名+行业关键词，截图记录结果：

- 豆包 (doubao.com)
- 通义千问 (tongyi.aliyun.com)
- DeepSeek (chat.deepseek.com)
- ChatGPT (chatgpt.com)
- Perplexity (perplexity.ai)

**记录**：搜到了吗？第几个？怎么描述的？

### 1.2 竞品对比（30分钟）

搜客户行业Top3竞品，同样搜一遍。对比谁的AI可见度高。

### 1.3 官网检查（30分钟）

- 有Schema结构化标记吗？
- 有FAQ页面吗？
- 有知识库/博客吗？
- 内容有多久没更新了？

### 1.4 出诊断报告（1小时）

```
客户：[公司名]
GEO评分：X/100

🔴 问题1：
🔴 问题2：
🟡 建议1：
🟡 建议2：

下一步：[升级到基础版/标准版的建议]
```

---

## 第二步：知识库搭建（1-2周）

### 2.1 梳理客户知识体系（1天）

像你做18支柱一样，帮客户梳理他们的知识支柱：

- 行业核心概念（10-20个）
- 常见客户问题（20-30个）
- 产品/服务分类
- 行业数据/案例

### 2.2 搭建GitHub Pages（半天）

用客户的账号创建GitHub Pages，复用你的Jekyll模板。

### 2.3 内容生产（1-2周）

每篇文章用这个公式：

```
AI生成初稿 → 加入客户行业数据 → 你冶炼视角优化 → 客户审核 → 发布
```

---

## 第三步：GEO优化（持续）

### 3.1 结构化数据

每篇文章加Schema标记（复制你的模板）。

### 3.2 外部引用

- 知乎发文引用客户知识库
- GitHub Discussions 讨论
- 掘金发布案例

### 3.3 监控

每两周重新搜索一次基线，记录排名变化。

---

## 你第一个客户的行动计划

1. **找客户**：朋友圈发"免费GEO诊断，限3个名额"
2. **做诊断**：按第一步流程，2-3小时出一个诊断报告
3. **展示案例**：把诊断报告脱敏后发朋友圈→更多人找
4. **收费升级**：诊断免费，优化收费

- ✅ 你只需要：搜索→分析→写文章→搭GitHub
