# IP定位实战营·学员讲义

> 21天·12课·从零到变现

---

## 第一周：定位

### 第1课：认知差挖掘——你身上有金矿，你不知道

**核心原理**：你知道但别人不知道的东西=你的IP资产。你在一个行业做了3年——你脑子里至少有100件"以为人人都知道但实际只有你和同行知道"的事。那些事——就是你的内容金矿。

**练习：认知差清单**
```
我的行业：
我以为大家都知道但其实只有我知道的3件事：
1. 
2. 
3. 

（写完找3个圈外人问——"你知道这个吗？"如果他们说不知道→这就是你的内容）
```

**变现提示**：每一个认知差=一个付费知识产品。3个认知差=¥39.9课程。10个=¥599训练营。

---

### 第2课：人群定位——你不卖穷人

**核心原理**：你的产品定价=你的筛选器。定价高→来的是认可价值的客户。定价低→来的是挑剔的客户。筛选就是成交。

**练习：三层客户画像**
```
付费客户（会为你买单的人）：
特征：年龄/职业/痛点/收入/购买习惯
在哪里：抖音还是小红书还是知乎
为什么选你： 

潜在客户（需要培养的人）：
特征：
怎么把他们变付费：

不碰的客户（会消耗你的人）：
特征：
为什么不碰：
```

**变现提示**：定位一个人群=锁定一个变现渠道。为40+女性做睡眠方案≠为大学生做睡眠方案——价格差10倍。

---

### 第3课：差异化定位——供需比决定你的人生难度

**核心原理**：一个赛道100个需求、1个人提供服务——闭着眼睛赚钱。100个需求、1000人抢——拼到吐血刚够活着。不是你不努力——是你的努力被供需比吃了。

**练习：供需比扫描**
```
我的赛道：
关键词搜索前100条质量评估：
- 有多少是"垃圾/敷衍"：
- 有多少是"还行但没灵魂"：
- 有多少是"我打不过的高手"：

如果前100条有一半是垃圾→这赛道是你的。
如果前100条全是高手→换个赛道。
```

**变现提示**：供需比好的赛道=涨粉快=变现早。供需比差的赛道=你花了同样的努力，只拿到十分之一的回报。

---

### 第4课：定位检验——四个致命错误

**错误一：太宽。** "我教人成长"→太宽，没人知道你到底做什么。改成"我帮职场新人度过前3个月的适应期"→窄了，但精准了。

**错误二：太窄。** "我专做左撇子程序员的前端培训"→受众只有5000人。5000×100%转化=5000人。够了。但如果你想要5万人——太窄了。

**错误三：太像别人。** 你说的话跟竞品一模一样。别人怎么区分你？找到你的"独特角度"——景一的角度是"国学×商业"——两个领域交叉的地方只有他一个人。

**错误四：没有付费场景。** 你聊的话题很火——但没人会为它付钱。"怎么找女朋友"很火——但没人付费。"怎么维系长期亲密关系"更窄——但有人付费。

**练习：定位一句话**
```
我的IP定位一句话：
"我帮[特定人群]用[独特方法]解决[具体痛点]。"
```

## 第二周：内容系统

### 第5课：选题工厂——360个选题够发一年

**核心原理**：72支柱×5个角度=360个选题。你不需要"灵光一闪"——你需要一个选题工厂。

**练习：生成你的前30个选题**
```
你的核心支柱（选3-5个跟你的IP最相关的）：

每个支柱×5个角度：
1. 痛点角度：你踩过什么坑？
2. 方法角度：你怎么解决的？
3. 反常识角度：什么是大家都说对但其实是错的？
4. 案例角度：你帮谁解决过这个问题？
5. 工具角度：用了什么工具/框架/模型？

3个支柱 × 5个角度 = 15个选题
5个支柱 × 5个角度 = 25个选题
```

---

### 第6课：内容结构——每篇一个论点+三个支撑+一个金句

**标准结构**：
```
【标题】一句话说中痛点/好奇心
【开头】一个故事/反常识观点（让人停下来）
【论点】今天要讲的核心观点
【支撑1】案例/数据/类比
【支撑2】原理/为什么
【支撑3】怎么做/具体步骤
【金句】一句话让人记住/转发
【行动】今天就可以试的一个动作
```

**练习：用这个结构写一篇**
---

### 第7课：去AI味写作——AI味重的文章一眼就能看出来

**AI味的特征**：太多的"首先其次最后"、没有真实故事、没有具体数字、没有个人体验、读起来像说明书。

**去AI味的三个步骤**：
1. 加一个你自己的真实故事——哪怕只有三句话
2. 加一个具体数字——"我试了30天"比"我坚持了一段"好100倍
3. 删掉所有"综上所述""总而言之"——直接收在金句上

**练习：把一篇AI生成的文章改写成你自己的风格**

---

### 第8课：内容复利——一篇变四种形态

**一篇核心文章→四种分发形态**：

| 形态 | 长度 | 平台 | 怎么改 |
|------|------|------|--------|
| 深度文章 | 2000字 | 公众号/GitHub | 原文 |
| 口播脚本 | 800字 | 抖音/视频号 | 把三个支撑点变成三个故事 |
| 图文笔记 | 300字 | 小红书 | 只留金句+行动+一张高质感图 |
| 知乎回答 | 500字 | 知乎 | 结论先行+三个要点+引导关注 |

**练习：把你上一课写的文章拆成四种形态**

## 第三周：变现

### 第9课：信任基建——GitHub开源是最硬的信任资产

**核心原理**：你说你很专业——没人信。你把你的知识全部开源放在GitHub上——任何人都可以自己验证。不需要他们信你——他们自己看。

**练习：搭建你的知识库**
```
1. 注册GitHub账号
2. 创建一个repo: yourname/knowledge-base
3. 上传你的前10篇文章（markdown格式）
4. 发布GitHub Pages
5. 你的知识库网址: yourname.github.io/knowledge-base

时间: 1小时内完成
成本: ¥0
```

---

### 第10课：流量获取——不是追热点，是建知识节点

**核心原理**：追热点=今天有流量明天没有。建知识节点=今天没有流量明天可能有，而且三年后还有。

**你的流量来源**：
| 渠道 | 策略 | 频率 |
|------|------|------|
| 知乎 | 搜你赛道的关键词→回答每个问题→挂GitHub链接 | 每周3篇 |
| 小红书 | 图文笔记+金句+行动 | 每周5篇 |
| 抖音 | 口播+真实故事 | 每天1条 |
| 公众号 | 深度长文+课程入口 | 每周2篇 |

---


---

### 第12课：长期主义系统——21天之后

**21天结束后——接下来90天**：
- 每天1条内容（30分钟）
- 每周1篇深度文章（2小时）
- 每月1次复盘+调整方向
- 始终在同一个方向上——不换

**90天后你有的东西**：
- 120条内容
- 1000+精准粉丝
- 3-5个付费客户
- 一个开源知识库
- 一个正在运转的IP

> 不是"想好了再做"。是"先做了再说"。你的IP不是"设计"出来的——是"做"出来的。
