## 一、GitHub 付费体系对标

### GitHub 官方收入模型

| 产品 | 定价 | 年收入（估算） | 景一能做什么 |
|------|------|-------------|------------|
| **Copilot** | $10-39/月 | ~$10亿+ | 知识版Copilot——景一AI咨询 |
| **Enterprise** | $21-39/用户/月 | 主力收入 | 企业知识管理方案 |
| **Actions** | 免费额度+按量付费 | 生态收入 | 自动化管道代部署 |
| **Packages** | 免费+按量 | 生态收入 | 付费内容分发 |
| **Codespaces** | 120h免费+$0.18/h | 生态收入 | 已配置 |
| **Sponsors** | 0%抽成 | 创作者收入 | 中国区被锁 |
| **Marketplace** | 5-25%抽成 | 平台收入 | 卖Actions/工具 |

### 关键洞察

GitHub 的收入密码不是卖产品——是**卖信任基础设施**。企业付费不是因为 GitHub 功能多，是因为"代码在 GitHub 上=专业"。

**景一的机会**：成为中文知识领域的 GitHub——你的知识体系=信任基础设施。

## 四、GitHub 作为信任引擎的不可替代性

| 竞争对手 | 他们有什么 | 景一有什么他们没有的 |
|---------|-----------|-------------------|
| 普通知识博主 | 抖音/小红书流量 | 开源可验证的知识体系 |
| 培训机构 | 线下渠道 | 零边际成本的自动化管道 |
| 咨询公司 | 方法论 | 免费公开=降低客户决策成本 |
| 出版社 | 发行渠道 | 自带流量和信任体系 |

**核心壁垒**：别人抄你一条内容只需要10分钟。抄18支柱+29篇精华+90册书库+自动化管道——没有一年半载不可能。而一年后你已经又多了一年的积累。
