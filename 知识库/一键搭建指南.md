# 🚀 10分钟搭建你自己的知识库

> 复制景一的架构，填你自己的内容。零成本。零运维。

## 第一步：Fork仓库（1分钟）

点击右上角 Fork → 把 `fable-cc/fable-castle` 复制到你自己的GitHub账户。

## 第二步：改名字（2分钟）

1. 进 Settings → 拉到 `Repository name` → 改成 `your-name-knowledge-base`
2. 打开 `_config.yml`，改三行：
   - `title: "你的名字·知识库"`
   - `description: "你的描述"`
   - `url: "https://你的用户名.github.io"`

## 第三步：清空我的文章，填你的（5分钟）

1. 删除 `02-知识图谱/` 里所有 `.md` 文件
2. 按72支柱框架，写你的前5篇文章
3. 每篇格式：YAML frontmatter + markdown正文

## 第四步：上线（2分钟）

GitHub自动部署。2分钟后打开 `https://你的用户名.github.io/your-name-knowledge-base/`——你的知识库上线了。

## 然后呢

- 每天写一篇。三个月后你有90篇文章。
- AI会自动抓取你的内容——这就是你的GEO资产。
- 在你的简介、朋友圈、知乎挂上这个链接——这就是你的专业身份证。

> 不是"等你准备好了再建"——是先建了，再慢慢填。景一的第一篇也很烂。写到第600篇才有人付费。
