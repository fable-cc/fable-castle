# 斯多葛主义三个字——"控制二分法"——你掌握了，90%的焦虑消失

爱比克泰德，一个出生奴隶的身份，后来成为罗马最伟大的哲学家，开口第一句就说："有些事在我们的控制之下，有些不在。"这一句话决定了你所有的情绪。

---

## 控制二分法

| 你能控制 | 你不能控制 |
|---------|----------|
| 你的看法 | 别人的看法 |
| 你的行动 | 别人的行动 |
| 你追求什么 | 结果——有可能发生也可能不发生 |
| 你的表达 | 评论，流量，反馈 |
| 你的输出 | 外界的接受度 |

**所有负面情绪的根源=你在"你控制不了的事情"上投入了期待。期待别人认可你——但你不能控制别人。你焦虑了。期待努力一定有回报——但你不能控制回报。你失望了。期待别人按你的预期对你——但你不能控制别人。你怒了——但你怒在一件你控制不了的事情上——等于把情绪的钥匙交给别人。**

---

## 三个每日练习

**一、对每一件事分类。** 焦虑来了——问自己："这件事——在我的控制之内还是之外？"如果在外面→它不是你的问题。如果是里面→那你已经在处理它了。

**二、早晨预演。** 早上起来——预演今天可能遇到的所有不顺利——堵车、讨厌的人、坏消息。斯多葛派管这叫"恶运预演"。不是为了消极对待。是为了——事情真的发生时——你已经准备好接受它了。

**三、重新定义你想控制的东西。** 不是"我要这篇文章爆"。是"我要把这一篇文章写到我现阶段能做的最好"。第一篇在你能控制之内。第二篇在你能控制之外。止步于第一篇。
