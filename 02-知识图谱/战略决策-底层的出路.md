# 底层出身的出路——不是"赢"——是"先不输"

原生家庭没有给你社会资源、人脉、启动资金。你的第一课——不是"怎么致富"——是"怎么不让他人夺走你已经有的东西。"你连第一步还没站稳——别人已经跟你要"贡献""付出""你是好人"——你先别做好人——先做"活下来的人"。

---

## 底层出身的五个误区

| 误区 | 真相 |
|------|------|
| "努力就够了" | 努力=基础。努力之外——还需要信息差——你的圈子没有人告诉你。知识=信息——你的72支柱——在给自己造信息。日后——别人想努力——但不知道往哪努力——你没白练 |
| "人际关系=人脉" | 在你没有价值的时候——人际关系=别人给你面子——不是资产。等你有了本事——人际关系自动变成了人脉——因为有人需要你。先有价值——然后人脉自然来 |
| "攒钱=财富" | 攒钱=保底。财富=资产×规模。光攒——不生——攒三十万——看病一次可能没了。想尽办法——建立哪怕最小的被动收入系统 |
| "跟着有钱人=变成有钱人" | 你可以近身观察——但不会变成他——他不是一个"培训班"。利用接触观察他的思维——不是期待他包你 |
| "运气=机会" | 运气=你准备得差不多了——忽然一个随机事件——你抓住了。没准备——那不是运气——是噪音——你听到了但用不上 |
