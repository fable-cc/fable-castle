# 赫拉克利特和老子同时说了一句话——一个在希腊，一个在中国

赫拉克利特，古希腊哲学家，公元前500年："你不能两次踏入同一条河流。"老子，中国道家，大约同时期："万物并作，吾以观复"——万物都在变化，我看它们循环往复。两个从未联系过的人。同一句话。同一个洞见。

## 万物流转

| 赫拉克利特 | 老子 |
|----------|------|
| 万物皆流 | 万物并作 |
| 对立统一——"向上的路也是向下的" | 阴阳——对立而统一 |
| "战争是万物之父" | "反者道之动"——对立的东西互相推动 |
| 火是基本元素——永远在变化 | 水——"上善若水"——适应一切而不被改变 |

**两个人都看到了核心真理：变化是唯一的常量。西方后来走向了原子论——万物由不变的微粒组成。东方走向了过程论——万物处在流动变化中。西方发现了"不变的分子"。东方发现了"永久的变化"。两个都是对的。但东方比西方早两千年意识到——"实体"只是暂时的——"过程"才是永恒的。**
