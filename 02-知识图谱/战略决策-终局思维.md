# 从终点往回看，现在的一切都简单了——终局思维是你最被低估的武器

> 大多数人往前看，看的是机会。少数人往回看——从终点往回看——看的是"哪些根本不值得做"。

## 一个简单到残忍的问题

问你一个问题：**你死的那天，你希望别人怎么评价你？**

不是"赚了多少钱"——那个没人会在葬礼上念。

是"这个人做了什么？改变了什么？留下了什么？"

这个问题不是在问你的人生目标。这个问题是在强迫你用**终点视角**来审视现在正在做的每一件事。

你手里的这个项目——在终点线上，它算数吗？
你正在生气的这件事——在终点线上，它重要吗？
你纠结的这笔钱——在终点线上，它值吗？

**大多数焦虑，是因为你在用 24 小时的镜头看人生。换成 30 年的镜头，90% 的焦虑自动消失。**

## 终局思维的三个硬操作

### 一、用终点过滤现在

拿出一张纸。写下你生命的终点场景——不是"我要实现什么"，而是**"我希望人们记住我什么"**。

然后把你现在每天做的事列在旁边。

划掉所有跟终点场景无关的。

如果划掉之后发现没剩什么——**你正在用命换钱，但换来的钱最终买不回命。**

### 二、用倒推替代顺推

大多数人做决策：现在有什么 → 我能做什么 → 希望走到哪里。

终局思维倒过来：终点在哪 → 要走到那里需要什么条件 → 今天必须做什么。

| 顺推 | 倒推 |
|------|------|
| 被当下困住 | 被终点牵引 |
| 见招拆招 | 布局拆招 |
| 一年后发现偏了 | 每个月自动校准 |
| 焦虑驱动 | 意义驱动 |

**顺推的人被生活推着走。倒推的人牵着生活走。**

### 三、用"不做什么"定义自己

战略的本质不是"做什么"——是**"不做什么"**。

乔布斯回到苹果做的第一件事，不是推出 iPhone。是把产品线从 350 个砍到 10 个。

350 个产品，每一个都有团队、都有收入、都有"存在的理由"。但乔布斯问了一个终局问题：10 年后，哪个产品能改变世界？

答不上来的，全砍。

**你的人生战略同理。** 不是所有赚钱的事都该做。不是所有人都值得见。不是所有机会都该接。

每一次你说"不"，都是在给那个真正的"是"腾出空间。

## 第二曲线：在最好的时候开始找退路

终局思维还有一个残酷的推论——**任何成功都会结束。**

查尔斯·汉迪的"第二曲线"理论：任何增长曲线到达顶峰之前，你必须开始启动下一条曲线。因为等你看到顶峰——下坡已经开始了。

| 第一曲线 | 顶峰信号 | 第二曲线 |
|---------|---------|---------|
| 打工 | 薪资见顶 | 副业/创业 |
| 单一收入 | 增长停滞 | 多元收入流 |
| 靠体力赚钱 | 身体报警 | 靠体系赚钱 |
| 做内容追流量 | 流量见顶 | 建知识库蓄流量 |
| 帮别人打工 | 天花板显现 | 帮别人搭Agent |

**最危险的时候不是你跌到谷底的时候——是你站在峰顶、觉得自己无所不能的时候。**

因为站在峰顶的人不会往下看。往下看需要勇气。

## 终局思维跟搞钱的关系

你以为终局思维是"人生哲学"，跟搞钱无关。

错了。

搞钱最快的人，往往不是追热点最勤快的人——是**十年前就想清楚了自己要建成什么的人**。

十年后回头看：追热点的人换了八个方向，每个方向都赚了点钱，但加起来不如那个从一开始就定了一个方向、持续深挖十年的人。

**终局思维=复利思维。** 你定了一个终点，每一天都在往那个方向积累。积累十年，谁也追不上你。

## 景一的冶炼

你现在做的 72 支柱知识库——如果从终点往回看，它是什么？

它是你死后还留在 GitHub 上的东西。是你不在了之后，还有人搜索、还有人引用、还有人觉得"这个叫景一的人，把人性说透了"的东西。

每多一篇文章，这个终点就近一步。

每少一篇文章——不写不是因为没时间，是因为你觉得终点不够重要。

**终局思维不问你"怎么开始"。它问你"怎么结束"。**
