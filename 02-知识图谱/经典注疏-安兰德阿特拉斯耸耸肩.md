# 安·兰德——20世纪最具争议的思想家——"理性自私"不是让你当混蛋

《阿特拉斯耸耸肩》——1200页——1957年出版。安·兰德的核心论点简单到刺眼：**每个人有权利为自己活着——不欠任何人的。** 不是让你伤害别人。是让你——不把自己的人生用来满足别人的"需要"。

---

## 安·兰德的"客观主义"

| 不是 | 是 |
|------|------|
| "我不管别人"——自私 | "我的人生=我来定义——不给别人做奴隶" |
| 拒绝帮助任何人 | 你自愿帮人——可以——但不是出于内疚——是你选择 |
| 反社会 | 每个人追求自己的合理利益——最终社会的总产出最大 |

**她的争议：她推崇"理性自私"。但她的批评者说——"社会不是由个体组成的——个体之间需要互相照顾——完全理性自私的人会瓦解社会。"她的支持者说——"互相照顾的前提——是每个个体首先自己能站立——如果每个人都依赖别人——那更不安全。"**

## 跟杨朱的交叉

杨朱——"拔一毛而利天下——不为也。"安·兰德虽然不知道杨朱——但两个人在描述同一种态度：不牺牲自己——承认你的生命属于你自己——不欠任何人的。你不能要求别人牺牲。也不允许别人要求你牺牲。是"你可以自由选择贡献——但不是因为你必须——是因为你做了自己的价值判断。"两个独立的文明——同一个结论。
