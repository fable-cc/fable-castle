# 毕达哥拉斯和《易经》的作者发现了同一个东西——"数"是宇宙的语言

毕达哥拉斯，古希腊，公元前500年："万物皆数。"音乐的和谐=数学比率。天体的运行=数学轨道。宇宙的本质=数学结构。同一年代——中国的《易经》体系——八卦=三爻×阴阳=2^3=8种状态。六十四卦=2^6=64种状态。本质=二进制数学——宇宙用数学写就。不是希腊人发现了"数学描述宇宙规律"——东方人独立发现了同一件事。

## 数学是连接东西方的桥梁

| 毕达哥拉斯 | 易经 |
|----------|------|
| 万物皆数 | 八卦=二进制——宇宙可以被数学描述 |
| 音乐的和谐=数学 | 64卦=64种状态——宇宙状态的数学编码 |
| 毕达哥拉斯定理：a²+b²=c² | 阴阳=对立互补——数学里的对偶性 |
| "数是万物的本源" | "太极生两仪"——一化为二——二的组合产生万物 |

**两个文明——独立发现——数学不是我们发明来描述宇宙的东西。宇宙本身就是数学结构。**
