# 做生意最赚钱的策略——恰好是"不做什么"

> 商业伦理不是道德绑架。是长期博弈的最优解。短期看诚信成本高。长期看——诚信是最便宜的。

## 义利之辨：儒家争论了两千年，商业给出了答案

孟子去见梁惠王。梁惠王第一句话："老先生你不远千里而来，给我带来什么利益？"

孟子回答："王何必曰利？亦有仁义而已矣。"——您何必一开口就谈利益？有仁义就够了。

两千年来，儒家一直用这句话来"压制商业"。但孟子不是这个意思。他是说：**只谈利益=短期博弈=所有人互相提防=交易成本爆炸=赚不到长远的钱。**

仁义=你的信誉=你不需要每次证明自己=交易成本最低=长期利润最大。

## 诚信复利：为什么"说到做到"是最被低估的商业策略

| 说到做到一次 | 说到做到十次 |
|------------|------------|
| 对方半信半疑 | 对方不需要你证明 |
| 每次交易都有验证成本 | 交易成本接近零 |
| 对方防着你 | 对方先付钱再等你交货 |
| 单次博弈 | 重复博弈 |

**商业世界最大的隐性成本不是房租、不是人工——是信任成本。诚信=把信任成本降到零。**

每次你兑现承诺——你往一个叫"信誉"的账户里存了一笔钱。这笔钱的利息极高：有一天你需要别人信任的时候——不需要解释。信誉账户里有足够余额。

## 什么不能卖

不是所有赚钱的事都该做。道理都懂，但"不该做"的边界在哪：

| 可以做 | 不能做 |
|--------|--------|
| 让客户花钱买价值 | 让客户花钱买伤害 |
| 信息差=信息不对称 | 信息差=欺骗 |
| 赚钱=帮了人 | 赚钱=害了人 |
| 客户可以离开 | 客户被套牢 |

**底线很简单：你做的事——你敢一字不差地公开告诉所有人吗？敢=应该没问题。不敢=有事。**

## 长期主义的商业回报

| 短期主义 | 长期主义 |
|---------|---------|
| 赚一次快钱 | 建一个信任品牌 |
| 客户来一次 | 客户来了不走 |
| 每次都要找新客户 | 老客户带新客户 |
| 收入波动大 | 收入稳定增长 |

**长期主义的秘密：你不需要聪明。你需要活得久、信誉好、让每一次跟你的交易都成为对方下一次找你的理由。**

**把你的"信誉账户"公开放在 GitHub 上。比任何销售话术都有说服力。**
