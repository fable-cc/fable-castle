# 你不是不会博弈——你是每天都在博弈，只是你不知道规则

> 你每一次说"好吧"、每一次沉默、每一次妥协——都在走一步棋。问题是你没意识到棋盘长什么样。

## 囚徒困境：为什么好人也会互相伤害

两个共犯被抓。分开审讯。检察官分别对每个人说同样的话：

- 如果你供出他，他不供你 → 你自由，他坐10年
- 如果他供出你，你不供他 → 你10年，他自由
- 如果你们互相供出 → 各坐5年
- 如果都不供 → 各坐1年

**两个人都选"不供"=总体最优。但每个人单独看——"供出对方"对自己最安全。** 结果：两个理性的人，都选了对自己最安全的选项，一起坐了5年。

这就是囚徒困境。它每天都在发生：

| 场景 | 囚徒困境 |
|------|---------|
| 合作 | 两个人都防着对方→合作不深入→双方都赚不到 |
| 关系 | 两个人都等着对方先付出→都不付出→关系凉了 |
| 团队 | 两个人都藏着资源不分享→团队不行了→一起丢工作 |

**解法**：重复博弈。如果你们知道还要跟对方打无数次交道——今天的背叛就会在明天被惩罚。所以合作是长期最优策略。

## 信号博弈：你做的每一件事都在传递信号

博弈论里有一个概念叫"信号发送"。不是你说什么——是你做什么。

| 信号 | 传递的信息 |
|------|-----------|
| 花三年建一个知识库 | "我不是来割韭菜的——沉没成本太大了" |
| 公开发表立场 | "我不会突然改口——改的成本太高" |
| 先给价值再收费 | "你先验货——我的东西不怕验" |
| 不砍价 | "我的定价是认真的——时间值钱" |

**最贵的信号是你不想发的——但你的行为已经在发了。**

你每次都妥协→信号："我的底线可以推。"你每次都迟到→信号："别人的时间不重要。"你每次都"再看看"→信号："我不确定自己要不要。"

## 承诺与威胁：哪个更有效

博弈论核心结论：**承诺比威胁更有效。**

威胁："你不做X，我就做Y"——对方会试探。如果你Y的成本很高，对方不信你会真的做。承诺："我会做X，不管你怎么做"——对方没得试探。你说了=你做。

| 威胁 | 承诺 |
|------|------|
| "你再这样我就走了" | "我的底线在这里。过了，我就走" |
| 等着看对方信不信 | 对方不需要猜——你说到做到 |
| 容易失效 | 逻辑自洽 |

**所以——别威胁。直接定底线。** 不用告诉对方"你再这样我就……"。直接说"这件事我不接受。如果发生，我会这么做。"然后照做。一次就够了。

## 景一的冶炼

博弈论不复杂。三个东西记住：

| 规则 | 含义 |
|------|------|
| 重复博弈 | 长期关系里，合作是最好的自私 |
| 信号 | 你做的每一件事都在告诉别人"你可以怎么对我" |
| 承诺>威胁 | 定底线，然后照做。不需要吼 |

**博弈论不是在教你算计别人。是在教你看清——别人一直在算你。你看清了，才能选择不被算。**
