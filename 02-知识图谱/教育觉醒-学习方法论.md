# 你不是学不会——是学校没教你怎么学

> 费曼说：如果你不能把一个东西用简单的话给别人讲明白——说明你自己还没真懂。

## 学校教你"学什么"，没教你"怎么学"

你花十几年学数学、语文、英语——没人教过你大脑怎么接收、储存、提取信息。

结果：大多数人毕业后——学习能力停留在高一水平。不是脑子不好。是从来没学会"学"这件事本身。

## 费曼技巧：讲给别人听

理查德·费曼，诺贝尔物理学奖得主。他的学习方法是：

| 步骤 | 操作 |
|------|------|
| 1 | 选一个你想学的概念 |
| 2 | 用最简单的话解释它——假装讲给一个完全不懂的人 |
| 3 | 讲到卡住的地方——那就是你还没真懂的地方 |
| 4 | 回去查、重新学——再讲 |
| 5 | 简化到能用一句话说清楚 |

**知识=你能解释给别人听并让人听懂的。不是你能认出来的。能认出来≠学会了。能讲明白=学会了。**

## 间隔重复：别临时抱佛脚

你的大脑不是硬盘——写进去就完了。大脑是肌肉——需要反复刺激才能记住。

艾宾浩斯遗忘曲线：学完一个小时之后——你忘了50%。一天之后——忘了70%。一周之后——忘了90%——除非你复习。

| 复习时机 | 效果 |
|---------|------|
| 学完立刻再学一遍 | 几乎没用 |
| 学完24小时后复习一次 | 记住率翻倍 |
| 再隔3天复习一次 | 记住率再翻倍 |
| 再隔一周复习一次 | 进入长期记忆 |

**间隔重复 = 在你快忘的时候复习。不是"重复越多越好"——是"在恰当的时机重复"。**

## 刻意练习：练你最不擅长的

大多数人的"练习"是——重复已经会的东西。这叫舒适区练习。不叫刻意练习。

刻意练习=找到你最差的那个环节——专门练它——练到不差为止——然后找下一个最差的。

| 普通练习 | 刻意练习 |
|---------|---------|
| 弹已经会的曲子 | 反复练那两小节总是弹错的 |
| 写已经擅长的文章 | 专门练不擅长的开头 |
| 做已经会做的题 | 只做刚好不会的题 |

**刻意练习=不舒服。不舒服=在进步。舒服=在维持。**

## 景一的冶炼

| 方法 | 一句话 |
|------|--------|
| 费曼技巧 | 讲给别人听→卡住→重新学→再讲 |
| 间隔重复 | 快忘的时候复习 |  
| 刻意练习 | 只练最差的环节 |

**学习的本质不是"记住更多"。是"用更好的方式对待自己的大脑"。**
