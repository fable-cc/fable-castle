# 选择悖论——越多的选择——越不幸福——越难——做出任何选择

巴里·施瓦茨——《选择的悖论》。超市——6 种果酱——更多人购买。24 种果酱——更少人买。不是因为——选择多不好——是——选择多——"我选的——会不会——不是——最好的——那个。"选完后——不幸福——因为——还在怀疑——"另一个——会不会——更好。"

## 选择悖论的日常

| 场景 | 效应 |
|------|------|
| 流媒体——整个目录——你不知道看什么 | 花四十分钟——滑——没选——最终看老剧——你明明喜欢——但未选新 |
| 约会 App——无数候选人 | 每个人都"下一个可能更好"——不认真——全不行——却都被筛 |
| 职业——做什么好——都是选项 | 你被可选项——淹没——迟迟不动——现在的工作——虽不满意——但"确定"——拖越久——越不敢动 |
