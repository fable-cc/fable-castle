# 为什么社会需要规则——荀子跟霍布斯隔了两千年，对人性得出了同一个结论

> 人性的底层代码，穿越时间和空间都不会变。荀子在战国说的那句话，霍布斯在 17 世纪的英国又说了一遍。

## 两个相隔两千年的人

荀子，战国末年，公元前 3 世纪。他说：**"人之性恶，其善者伪也。"** 人性本恶。善是后天训练出来的。

霍布斯，英国，公元 17 世纪。他说：**"自然状态下，人的生活是孤独、贫穷、肮脏、野蛮和短暂的。"** 没有规则，人就是野兽。

两千年的间隔。一个在东方，一个在西方。一个用古汉语，一个用英文。但他们对人性的判断——**一模一样。**

## 荀子的人性论

荀子不是骂人。他是在冷静地分析。

"人生而有好利"——人生来就想要对自己有利的东西。
"生而有疾恶"——人生来就会嫉妒和讨厌别人。
"生而有耳目之欲"——人生来就有感官欲望。

这三个东西，不加约束 → 争夺 → 混乱 → 暴乱。

荀子说的"性恶"，不是说人本质上是坏的。是说**人的出厂设置如果不经过文化训练，会走向对他人的伤害。**

所以荀子提出"化性起伪"——用礼法、教育、制度来约束和引导人性。

**"伪"不是虚伪。是人为的、后天的、文明的东西。** 人性是矿石，礼法是冶炼。这跟你的 72 支柱冶炼逻辑一模一样。

## 霍布斯的利维坦

霍布斯的起点跟荀子一样——人性很危险。

在没有政府的"自然状态"下，每个人都是自己的法官和执法者。你认为别人冒犯了你，你就自己动手。你怕别人先动手，你就先动手。

结果：**所有人对所有人的战争。**

所以霍布斯说：人们需要一个"利维坦"——一个强大的、被所有人承认的主权者——来维护秩序。人们自愿放弃一部分自由，换取安全和规则。

**社会契约 = 交出私刑权 → 换取公共安全。**

## 同一个结论

| | 荀子 | 霍布斯 |
|------|------|------|
| 时代 | 战国（前 3 世纪） | 英国内战（17 世纪） |
| 人性判断 | 人性本恶 | 自然状态=战争状态 |
| 解法 | 礼法+教育 | 社会契约+利维坦 |
| 核心洞察 | 规则驯化人性 | 制度约束暴力 |
| 最终目标 | 秩序 | 和平 |

**两个人相隔两千年，背景完全不同，却得出了同一个结论——人性必须被制度驯化，否则社会就是丛林。**

## 为什么这对你很重要

荀子和霍布斯不是在写学术论文。他们在回答一个所有人都要面对的问题：

**"在这个世界上，你凭什么相信一个陌生人不会伤害你？"**

答案是：不是因为他善良——是因为制度。

不是因为你的客户诚实——是因为合同。
不是因为你的员工忠诚——是因为体系。
不是因为你的合作伙伴可靠——是因为双方都有退出成本。

**信任不是靠人品建立的。是靠制度建立的。人品是变量，制度是常量。**

## 景一的冶炼

| 维度 | 道理 |
|------|------|
| 做企业 | 不要靠员工的自觉——靠流程 |
| 做关系 | 不要靠对方的承诺——靠双方都有退出成本 |
| 做合作 | 不要靠"我相信他"——靠"他违约的代价大于履约" |
| 做IP | 不要靠粉丝的忠诚——靠你的知识库不可替代 |

**荀子和霍布斯的共同遗产：不要指望人性变好。设计制度让人性变好。**
