# 你不想谈死亡，但死亡每天都在塑造你的每一个决定

> 海德格尔说：人只有面对死亡的时候，才开始真正地活着。

## 你逃避的东西正在决定你

死亡是这个世界上最被回避的话题。但它是唯一100%确定的事。

你不敢辞职——因为"万一失败了呢"背后的恐惧，是死亡在说话：时间有限，我怕浪费。你不敢表白——因为"万一被拒绝呢"，也是死亡在说话：我怕在有限的时间里承受痛苦。你拼命赚钱——因为"我要趁年轻多攒点"，更是死亡在说话：我知道我不会永远有这个能力。

**你所有最在意的决定——背后全是死亡的影子。但你不看它。**

## 向死而生

海德格尔发明了一个词：**"向死而生"**。不是悲观——是清醒。

大部分人活在"跟死亡无关"的幻觉里——好像自己永远有时间。明天再说。下次再说。等准备好了再说。

但海德格尔说：**你一旦真正接受了"我会死"这个事实——你的生命质量会瞬间提高。**

| 回避死亡 | 面对死亡 |
|---------|---------|
| "以后再……" | "如果现在就做呢？" |
| 被琐事填满 | 自动过滤不重要的事 |
| "别人怎么看我"很重要 | "别人怎么看我"不重要了 |
| 拖延 | "我没有无限的时间" |
| 怕这怕那 | 怕的东西变少了——因为你怕的最大那个已经接受了 |

**面对死亡不是让你抑郁。是让你清醒。**

## 遗产——你留下什么

"遗产"听上去是有钱人的词。但遗产=你死之后还在的东西。

| 遗产类型 | 例子 |
|---------|------|
| 基因 | 你的孩子 |
| 作品 | 你写的文章·你建的知识库 |
| 影响 | 你改变过的人的行为和想法 |
| 记忆 | 别人想起你时的感觉 |

**你不需要有钱才有遗产。你需要的是——你在这个世界上留下了什么东西，在你不在之后，还在。**

72支柱知识库。你死之后，它还在 GitHub 上。有人搜到、有人读到、有人被改变。这是遗产。

## 无常——佛教和存在主义的交叉点

佛教说"诸行无常"——一切都在变，一切都会过去。

存在主义说"人是自由的"——你随时可以重新定义自己。

**两条线在死亡这里交汇：因为一切都会过去，所以你有自由让每一个过去的东西不再是你的枷锁。**

昨天的失败——会过去。过去的东西。今天的成功——会过去。过去的东西。你最恨的人——会死。你也一样。那你还剩多少时间用来恨？用来后悔？用来等？

**无常不是坏事。无常=你可以重来。**

## 景一的冶炼

你敢不敢今天做一件事——**给十年后的自己写一句话。**

不是目标。不是计划。是一句话——如果十年后你回头看，你希望自己"没有浪费这十年"。

那句话写下来——它就是你的向死而生。不是让你恐惧。是让你知道自己最在意什么。

**而知道自己最在意什么的人——不需要别人告诉他该怎么活。**
