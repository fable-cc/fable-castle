# 刘黑闼逃跑时带走了200人——剩下两万人是"炮灰"。你的职场同理

李世民对刘黑闼。相持两个月。刘黑闼粮食耗尽→计谋失效→体力不支。于是高层开会——王小胡和刘黑闼先逃——200多核心圈的人跟着——剩下两万人不知道将领跑了——还在死战——一万多被杀——几千人被淹死。

**你不是忠诚。你是不知道老板已经跑了。**

## 成为炮灰的六大标志

| 类别 | 标志 | 怎么识别 |
|------|------|---------|
| 组织层 | **资源耗尽** | 公司有没有钱——从报销速度、工资准时度、招聘冻结——就能推出来。等公开宣布"资金困难"——已经是最后一刻了 |
| 组织层 | **智能耗尽** | 连续几个项目都失败——策略换了又换——没有一个见效——说明上层已经没有有效方案了 |
| 组织层 | **体力耗尽** | 靠加班硬撑——但对手资源雄厚——消耗战你没有优势——"再坚持一下"=让你做炮灰 |
| 个人层 | **非核心人物** | 收缩的时候——核心圈先走——剩下的走不了。你是不是每次开会都在决策圈外——每次关键信息都是事后才知道——如果是——你不是核心 |
| 个人层 | **无准确信息** | 高层已经暗中转移——但你还不知道——因为你"只关注业务——不关注组织"。信息差=生存差 |
| 个人层 | **无特殊技能** | 一万多人被杀——但游泳高手可以游走——武艺强者可以投降。你有不可替代的能力——你就有退路 |

## 反制——不做炮灰的四件事

**一、跟踪财务信号。** 不是偷钱。是看报销周期、工资准时度、招聘冻结。组织在崩之前——财务会先发出信号。

**二、审视组织能力。** 连续几个项目都失败——不是"运气不好"——是组织的智力策略已经无用了——你个人再努力也扭转不了。

**三、靠近核心圈。** 每次决策后问自己——"我在这个决策之前——还是之后——知道这件事"。如果在之后——你离核心圈还有距离——想办法往内走——或换一个能在核心的组织。

**四、打造不可替代的技能。** 不是让你"更努力"。是让你拥有一个——"你走了——组织这个功能就断了"的技能。这是你唯一的安全网。
