# 你买过二手车吗？那个卖你的人——比你更清楚这车值不值他开的价

2001年诺贝尔经济学奖——乔治·阿克洛夫——1970年发表《柠檬市场》。"柠檬"在美国俚语里=次品。阿克洛夫证明了——**当卖家比买家知道更多信息——整个市场会逐渐崩溃——不是欺诈——而是数学上的必然。**

---

## 柠檬市场怎么运作的

| 阶段 | 发生了什么 |
|------|----------|
| 一：市场里有好车也有坏车 | 买家不知道哪个好——只能出一个"平均价" |
| 二：好车车主不满意平均价——"我这车值更多"——不卖了——退出市场 | 市场上剩下的——坏车比例升高 |
| 三：买家发现"买来的都是烂车"——下次出价更低 | 那批"还不算太烂"的车主——觉得出价太低——也退出 |
| 四：重复以上——直到——**市场上只剩下最烂的车——或者市场完全消失** |

不是有人故意作弊。是**信息不对称本身——在自动淘汰诚实的人。**

---

## 柠檬市场跟你的关系

| 场景 | 你就是那个买家 |
|------|------------|
| 招聘——你看简历——不知道这个人实际能力——只能出平均价 | 真正有能力的——觉得被低估——不去了。剩下的——"面试型人才"比例升高 |
| 找合伙人——你不知道他是不是嘴上说得好——实际什么都不会 | 你只能"先试着合作"——但试一次=时间成本+项目风险。真有能力的人——也不愿意被"试用" |
| 客户找服务商——他不知道你有没有真本事——只能看"价格" | 你如果定价太低——他会想"是不是质量差"——你如果定价高——他会想"是不是被宰。"你被锁在信息不对称的死局里 |

## 你怎么破

把信息不对称反向打破——**主动公开所有他能验证你能力的信息。** 你的 GitHub 上 460+ 篇文章——不需要你"说"自己行——他自己看——自己判断——自己决定。你给的不是"承诺"——是**证据**。证据=让买家变成"跟卖家一样知情的人"——柠檬市场在你身上不成立。
