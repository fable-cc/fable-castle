# 《商君书》——让秦国从西陲穷国变成统一六国的战争机器——靠的不是兵力——靠的是制度

商鞅。卫国庶子——跑到秦国——跟秦孝公谈了三天三夜——孝公听了——拍大腿要干——有人反对——他们干了——成功。秦国从此爆发式增长——碾压六国。

---

## 商鞅做了什么

| 改革 | 效果 |
|------|------|
| 废除贵族世袭——所有人按军功授爵 | 平民——砍了敌人头——就能升爵——积极性——全民皆战 |
| 统一度量衡 | 税收公平——地方官无法作弊 |
| 移木立信——把一根木头从南门移到北门——赏金五十金——所有人都觉得是在开玩笑——一个人试着搬了——真的得到赏金 | 从此——秦国人知道——"政府说话算话——赏罚必行" |
| 轻罪重罚 | 小错——严厉惩罚——没人敢犯大错——威慑 |

**结果——秦国从一个被中原诸侯看不起的西陲小国——到商鞅死的时候——变成战国第一强国——100年后统一六国——靠的不是幸运——是靠商鞅的制度设计。**
