# 马斯克——"快速做决定——比做出完美的决定——更重要"

"如果你做一个决定需要花一年——你早就被淘汰了。"SpaceX 的节奏——今天发现问题——今天讨论——今天决定——今天开始执行。不要等"完美的数据"。

## 决策速度三原则

| 慢 | 快 |
|------|------|
| 等所有信息——"万一还有我不知道的" | 有足够信息就动——"错了——可以很快改" |
| 怕错——反复开会——反复论证 | "错一次的成本——远低于等待中被竞对甩开的成本" |
| "这件事需要老板批"——层层上报 | "在你权限范围内的——自己决定——错了算你的——但你不能不决定" |

**马斯克的决定速度=SpaceX 的核心竞争力。"错——很快会暴露——暴露了就改。你犹豫的时间——对手已经追上来了。"**
