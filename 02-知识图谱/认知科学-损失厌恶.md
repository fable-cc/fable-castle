# 损失厌恶——你"害怕失去"的程度——是"渴望得到"的两倍

卡尼曼的前景理论：**人对"失去"的感受强度——大约是"得到同样东西"的两倍。** 你丢100块的痛苦——比捡到100块的快乐——强两倍。不是你的错——是你的大脑——在进化中默认——"失去=死亡风险——必须放大。"

## 损失厌恶的日常陷阱

| 陷阱 | 表现 |
|------|------|
| 持有亏损——不卖——因为卖出=承认损失——"也许还能弹回来。" | 不是理性——是损失厌恶绑架——非要赢回——通常输更多 |
| 拒绝尝试新事物——怕"失去现有"——不计可能得到更多 | 你保住旧的——但也失去了新的——而新的——可能比旧的值钱十倍 |
| "免费试用"让你"不取消"——因为取消=失去已经得到的东西——尽管你不需要 | 你不是需要——是怕失去——"我得留着——万一。" |
